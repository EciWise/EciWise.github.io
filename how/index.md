# How it works

Technical guides and architecture diagrams for each ECIWise service.

## Services

- [AI Service](./ai-service.md) — AI-based academic recommendations and assistance.
- [Auth Service](./auth-service.md) — Authentication, OAuth 2.0 with Google, and JWT issuance.
- [Community Service](./community-service.md) — Forums, community, and student communication.
- [Game Service](./game-service.md) — Real-time multiplayer game server (WebSocket, Go).
- [Gamification Service](./gamification-service.md) — Points, achievements, and leaderboards.
- [Materials Service](./materials-service.md) — Academic PDF repository with AI validation and cloud storage.
- [Notifications Service](./notifications-service.md) — Real-time notifications.
- [Study Service](./study-service.md) — Interactive study module and practice.
- [Talk Service](./talk-service.md) — Real-time chat with conversations, reactions, and attachments.
- [Tutoring Service](./tutoring-service.md) — Virtual and in-person tutoring sessions.
